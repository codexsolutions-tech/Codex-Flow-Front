# Sobre

O **CodEx Flow** é uma plataforma de gestão empresarial desenvolvida para oferecer velocidade, simplicidade e produtividade no dia a dia das empresas.

Criado com foco em performance e experiência do usuário, o sistema reúne diversas soluções administrativas em uma interface moderna, intuitiva e altamente escalável.
